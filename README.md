# Sumit Deshmukh — Portfolio

Premium, recruiter-focused portfolio built with React, Vite, and Framer Motion.

## Quick start

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Before deploying

1. Replace `public/resume.html` with your PDF at `public/resume.pdf` and update `resumePath` in `src/data/portfolio.js`.
2. Update project GitHub and live demo URLs in `src/data/portfolio.js`.
3. Confirm your GitHub username in `personal.github` if different.

## Deploy to Vercel

Connect this repository to Vercel or run `vercel` from the project root. Build command: `npm run build`, output: `dist`.

## Tech stack

- React 19 + Vite
- Framer Motion
- React Icons
- CSS custom properties (design tokens)
